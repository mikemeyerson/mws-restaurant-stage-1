# Mobile Web Specialist Certification Course
---
#### _Three Stage Course Material Project - Restaurant Reviews_

## How To Run

Clone this repo and the [Stage 3 API](https://github.com/udacity/mws-restaurant-stage-3) repo.

*Server*
- `npm install`
- `node server`

*Client*
- `npm install`
- `npm run build`
- `npm run start`
